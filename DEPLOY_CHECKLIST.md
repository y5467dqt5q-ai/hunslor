# ✅ Чеклист для деплоя на Railway

- [ ] Все изменения закоммичены в Git
- [ ] Код запушен в GitHub репозиторий
- [ ] Создан аккаунт на Railway.app
- [ ] Railway подключен к GitHub

## Настройка Railway:

- [ ] Создан новый проект на Railway
- [ ] Подключен GitHub репозиторий
- [ ] Добавлена PostgreSQL база данных
- [ ] Добавлены все переменные окружения:
  - [ ] `DATABASE_URL` (автоматически от Railway)
  - [ ] `JWT_SECRET` (сгенерирован случайный секрет)
  - [ ] `TELEGRAM_BOT_TOKEN`
  - [ ] `TELEGRAM_ADMIN_ID`
  - [ ] `OPENAI_API_KEY`
  - [ ] `NODE_ENV=production`

## Команды для проверки:

```bash
# Проверить логи
railway logs

# Подключиться к базе данных
railway connect postgres

# Проверить переменные окружения
railway variables
```

## Если что-то не работает:

1. Проверьте логи в Railway Dashboard
2. Проверьте переменные окружения
3. Проверьте, что база данных создана
4. Проверьте, что все зависимости установлены
5. Проверьте, что build прошел успешно
