Следующие шаги для Railway:

1. Откройте: https://railway.app/project/a6111262-b4c7-468f-97e6-099305db819c

2. Нажмите "+ New" → "GitHub Repo" → выберите "y5467dqt5q-ai/hunslor"

3. Добавьте PostgreSQL: "+ New" → "Database" → "Add PostgreSQL"

4. В Settings → Variables добавьте:
   JWT_SECRET=ваш-секрет-минимум-32-символа
   TELEGRAM_BOT_TOKEN=ваш-токен
   TELEGRAM_ADMIN_ID=ваш-id
   OPENAI_API_KEY=ваш-ключ
   NODE_ENV=production

5. Дождитесь завершения деплоя

6. Получите URL вашего приложения
